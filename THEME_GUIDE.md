## How to Use Themes

### As a Drinker (the-drinker)

#### 1. Explore by Theme
- Open the **Filter Panel** (funnel icon)

#### 2. View Daily Content
- Navigate to the **Daily** tab (sparkle icon)
- Each day features one randomly-selected theme
- Read the quote, story, and cocktail recipe

- Content updates every 24 hours

Each theme's daily content includes:

## Technical Implementation

### For Developers
Themes are implemented through:

**Color Schemes** (`/src/lib/theme-config.ts`):
```typescript
THEME_COLOR_SCHEMES[theme] = {
  primary, secondary, accent,
  background, cardBg,
  textPrimary, textSecondary,
  borderColor, glowColor
}
```

**Daily Content** (`/src/lib/daily-content-service.ts`):
- Uses OpenAI GPT-4 for generation
- Caches per day to avoid redundant calls
- Fallback content if API fails
- Theme-specific prompts

**Visual Application**:
- CSS custom properties per theme
- Inline styles for dynamic colors
- Gradient backgrounds
- Border/glow effects

**Data Structure**:
```typescript
type DrinkingTheme = 
  | 'famous-drunks'
  | 'literary'
  | 'archetypal'
  | 'prohibition'
  | 'ancient-rome'
```

## FAQ

**Q: Can a venue have multiple themes?**
A: Yes! Bourbon & Branch has Famous Drunks, Literary, and Prohibition.

**Q: How often does daily content change?**
A: Every 24 hours at midnight (local time).

**Q: Can I see all themes' daily content?**
A: Currently one per day, but you can manually trigger generation for others.

**Q: Do themes affect recommendations?**
A: Yes, selected themes influence venue and event suggestions.

**Q: Can bartenders create custom themes?**
A: Not yet, but it's on the roadmap!

**Q: Are the historical facts accurate?**
A: Yes, AI-generated content is based on historical records, though you should verify surprising claims independently.
